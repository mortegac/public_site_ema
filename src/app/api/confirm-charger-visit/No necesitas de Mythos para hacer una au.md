No necesitas de Mythos para hacer una auditoría de seguridad de tu proyecto, lo puedes hacer con Claude Code y Opus 4.8 ahora mismo.

1. Entra a al proyecto y abre claude code
2. /effort y escoge el nivel ultracode
3. En el prompt escribe, Haz una auditoría de seguridad completa del proyecto ultracode (es importante volver a mencionar ultracode aunque ya sea parte del effort, de esa forma se activan los workflows). Verás que la palabra se resalta y en el status bar dice dynamic workflow requested. 
4. Ejecútalo y espera media hora (o más)
5. Listo, ahora procede a parchar decenas de problemas.

Necesitarás de un plan de 100 USD o mayor porque consume un montón de tokens. En las pruebas que hice me comía aproximadamente el 50% de uso diario en cada worflow y eran proyectos mediano solamente.
